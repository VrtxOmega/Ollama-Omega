# Changelog

## [1.0.0] — 2026-04-15

### Added
- Initial release of **Ollama-Omega** MCP server
- 6 MCP tools: `ollama_health`, `ollama_list_models`, `ollama_chat`, `ollama_generate`, `ollama_show_model`, `ollama_pull_model`
- Ollama Cloud support (Qwen 3.5 397B, DeepSeek V3.1 671B, etc.)
- Local model support (any pulled Ollama model)
- 300s configurable timeout for cloud model inference
- Structured logging to stderr
- SSRF mitigation (`follow_redirects=False`)
- Safe JSON parsing with `_safe_json()` wrapper
- Argument validation with `_validate_required()` guard
- Global singleton `httpx.AsyncClient` for connection reuse
- Sanitized error responses (no stack trace leakage)

### Origin
- Server code generated by **Qwen 3.5 397B** (Ollama Cloud)
- Self-audited by the same 397B model (11 issues found)
- Hardened with 7 fixes applied by the Antigravity IDE agent
